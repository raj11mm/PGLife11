# 🏠 PGLife11

PGLife11 is a simple and responsive web application designed to help users find PGs (Paying Guest accommodations) and hostels easily. It provides a clean interface to browse listings and view accommodation details.

---

## 🚀 Features

- Browse PG and hostel listings  
- Clean and responsive UI  
- Simple navigation and layout  
- Easy-to-use interface for users  

---

## 🛠️ Tech Stack

- HTML  
- CSS  
- JavaScript
